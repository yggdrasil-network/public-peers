# Latvia Peers

Add connection strings from the below list to the `Peers: []` section of your
Yggdrasil configuration file to peer with these nodes.

* Latvia, Shared 10Gbps dual-stack, Operated by strcat.su
  * `tls://vpn1.strcat.su:8443`
  * `tcp://vpn1.strcat.su:8888`
  * `quic://vpn1.strcat.su:5353
  * `ws://vpn1.strcat.su:8080`
