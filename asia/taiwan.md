# Taiwan Peers

Add connection strings from the below list to the `Peers: []` section of your
Yggdrasil configuration file to peer with these nodes.

### Taichung

* Taipei, Taiwan, FreeBSD nano EC2 on AWS ap-east-2, operated by [Alexa](https://lxak.net) (dual-stack)
  * `tcp://ygg-tw.lxak.net:80`
  * `tls://ygg-tw.lxak.net:443`

* Taichung, Taiwan, Homelab operated by [CharlieMoomoo](https://github.com/charlie-moomoo) (dual-stack, IPv6 tunneled via STUIX and might +5ms)
  * `tls://cowgl.tech:1337`
