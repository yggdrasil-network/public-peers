# Taiwan Peers

Add connection strings from the below list to the `Peers: []` section of your
Yggdrasil configuration file to peer with these nodes.

### Taichung

* Taichung, Taiwan, Compute Instance on Google asia-east1, operated by [Alexa](https://goblin.technology/@lxak) (dual-stack)
  * `tcp://ygg-tw.lxak.net:8879`
  * `tls://ygg-tw.lxak.net:8880`
  * `quic://ygg-tw.lxak.net:8881`
